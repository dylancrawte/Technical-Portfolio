Go4-3

React Native Typescript Zustand Node JS Express Product Management

AWS Bedrock Copilot Studio

Automated Newsletter

Power Automate Python

When i first joined IAG, they had me manually filling out the monthly newsletter. So, I designed a system that automatically retrieves news articles and generates a report of summaries. It features weekly, automated emailing, a dynamic date filter for the content, and embedded hyperlinks to the articles in the titles.

This project is designed and maintained by myself. It began as a python script that utilised a news API, but has since been changed to a more effective power automate solution.

Computational Neuroscience

Python Data analysis

At University, I studied Neuroscience, and in my master's year, my dissertation focused around EEG and computational neuroscience. The successful project involved working with a stimulation pattern that aligned with the peaks and troughs of a participants EEG signal (closed-loop stimulation), as a treatment for neuro-muscular conditions. The project involved work using Python to design and edit the stimulation pattern, as well as analysing the data collected from the participant.
